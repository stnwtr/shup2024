# Shared Memory - Einstieg

## Bauen

Dieses Programm kann mit `gcc -Wall ../../common/common.c shm.c -o shm.out` gebaut werden.

## Ausführen

Das Programm kann nach dem Bau mit `./shm.out <anzahl> <seed>` ausgeführt werden.
